利用規約

最終更新日：2026年1月1日

この利用規約（以下「本規約」）は、ネザーWEBサービス（以下「当社」）が提供するネザーID（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただく前に、本規約をよくお読みください。本サービスにご登録いただくことで、本規約に同意いただいたものとみなします。

第1条（定義）

本規約において使用する用語の定義は以下のとおりです。

  1. 「本サービス」とは、当社が提供するネザーIDおよびこれに関連するサービス全般を指します。
  2. 「ユーザー」とは、本規約に同意のうえ、本サービスの利用登録を行った個人または法人を指します。
  3. 「登録情報」とは、ユーザーが本サービスの利用登録または利用にあたって提供した情報を指します。
  4. 「コンテンツ」とは、本サービス上においてユーザーが投稿・送信・掲載したテキスト、画像、その他一切の情報を指します。

第2条（利用登録）

  1. 本サービスの利用を希望する方は、当社の定める方法に従い、本規約に同意のうえ利用登録を申請するものとします。
  2. 当社は、利用登録の申請者が以下のいずれかの事由に該当する場合、登録を拒否することがあります。
    1. 虚偽の情報を提供した場合
    2. 本規約に違反したことがある場合
    3. 反社会的勢力等に該当する場合
    4. その他、当社が不適切と判断した場合
  3. ユーザーは登録情報を常に最新かつ正確な状態に保つ責任を負います。
  4. 本サービスのアカウントは一身専属であり、第三者への譲渡・貸与は禁止します。

第3条（アカウントの管理）

  1. ユーザーは、自己の責任においてパスワード等のアカウント情報を管理するものとします。
  2. ユーザーは、アカウント情報を第三者に開示・共有・貸与してはなりません。
  3. アカウント情報の不正利用により生じた損害について、当社は一切の責任を負いません。
  4. アカウント情報の不正利用を発見した場合、ユーザーは直ちに当社へ報告するものとします。

第4条（禁止事項）

ユーザーは、本サービスの利用にあたり以下の行為を行ってはなりません。

  1. 法令または本規約に違反する行為
  2. 当社または第三者の権利（著作権・商標権・プライバシー権等）を侵害する行為
  3. 虚偽の情報を登録・送信する行為
  4. 本サービスの運営を妨害する行為（スパム・不正アクセス・サーバー攻撃等）
  5. 他のユーザーや第三者を誹謗中傷し、または名誉・信用を傷つける行為
  6. 反社会的勢力への利益供与その他一切の協力行為
  7. 当社が別途禁止として指定した行為
  8. 上記各号に準じる行為またはそのおそれのある行為

第5条（サービスの変更・停止・終了）

  1. 当社は、ユーザーへ事前通知することなく、本サービスの内容を変更することがあります。
  2. 当社は、以下の事由が生じた場合、本サービスの提供を一時停止または中断することがあります。
    1. システムの保守・点検を行う場合
    2. 天災・停電・不可抗力による障害が生じた場合
    3. その他、当社が停止を必要と判断した場合
  3. 当社は、相当な予告期間を設けたうえで、本サービスを終了することがあります。
  4. 前各項によりユーザーに損害が生じた場合でも、当社は一切の責任を負いません。

第6条（知的財産権）

  1. 本サービスに含まれるすべてのコンテンツ（テキスト・画像・デザイン・ソフトウェア等）の知的財産権は、当社または正当な権利者に帰属します。
  2. 本規約の許諾する範囲を超えた複製・転載・改変・二次利用等の行為は禁止します。
  3. ユーザーが投稿・送信したコンテンツの著作権はユーザーに帰属しますが、当社はサービス運営・改善に必要な範囲でコンテンツを利用できるものとします。

第7条（免責事項）

  1. 当社は、本サービスの内容・品質について明示または黙示を問わず保証しません。
  2. 当社は、本サービスの利用によりユーザーまたは第三者に生じた損害について、当社の故意または重大な過失による場合を除き、責任を負わないものとします。
  3. 当社は、第三者が提供するサービスおよびリンク先ウェブサイトの内容について、いかなる責任も負いません。

第8条（退会）

  1. ユーザーは、当社の定める手続きに従い、いつでも退会（アカウント削除）を申請できます。
  2. 退会後は登録情報および利用データが削除され、復元できない場合があります。
  3. 退会後も、当社が法令に基づき保持する義務のあるデータについては引き続き保管します。

第9条（本規約の変更）

  1. 当社は、必要と判断した場合に本規約を変更することができます。
  2. 変更後の規約は、当社ウェブサイト上での掲示をもってユーザーへ通知するものとします。
  3. 変更後に本サービスを継続利用した場合、ユーザーが変更後の規約に同意したものとみなします。

第10条（準拠法・管轄裁判所）

  1. 本規約の解釈および適用は、日本法に準拠します。
  2. 本サービスに関して紛争が生じた場合、当社の所在地を管轄する裁判所を専属的合意管轄裁判所とします。
