NetherID Custom Backend API (1.0.0)

Download OpenAPI specification:

NetherIDの独自バックエンド（Laravel）が提供するAPI仕様。認証はOry KratosのセッションCookieを利用します。

ダッシュボード初期情報取得

マイページに必要な「お知らせ」と「サービス連携状況」を一括取得します。KratosセッションCookieが必須です。

Authorizations:
cookieAuth

Responses

Production Server

https://api.netherid.example.com/dashboard/init

Local Development Server

http://localhost:8000/dashboard/init
